### 2.2 安装 Node.js

OpenClaw 基于 Node.js 开发，需要 Node.js v20+。

**方式一：官网下载（推荐新手）**

1. 访问：https://nodejs.org/
2. 下载 LTS 版本（长期支持版）
3. 运行安装程序
4. 一直点 Next 即可

**验证安装：**
```bash
node --version
# 应该显示 v20.x.x 或更高
```

**方式二：使用 nvm（推荐开发者）**

nvm（Node Version Manager）可以管理多个 Node 版本。

```bash
# Windows 安装 nvm
# 访问：https://github.com/coreybutler/nvm-windows/releases
# 下载 nvm-setup.exe 安装

# 安装 Node.js 22
nvm install 22
nvm use 22

# 验证
node --version
```

### 2.3 安装 npm 包管理器

npm 会随 Node.js 一起安装，验证：

```bash
npm --version
# 应该显示 10.x.x 或更高
```

## 三、安装 OpenClaw

### 3.1 全局安装 OpenClaw

打开命令行（PowerShell 或 CMD），运行：

```bash
npm install -g openclaw
```

**安装成功标志：**
```
added 150 packages in 30s
```

### 3.2 验证安装

```bash
openclaw --version
```

应该显示版本号，如 `openclaw 1.x.x`

```bash
openclaw --help
```

可以看到所有可用命令。


## 四、配置 AI 模型

### 4.1 选择模型提供商

OpenClaw 支持多种 AI 模型：

| 提供商 | 模型 | 免费额度 | 推荐度 |
|--------|------|----------|--------|
| **百度智能云** | 千问 Qwen3.5 | 有 | ⭐⭐⭐⭐⭐ |
| **阿里云** | 通义千问 | 有 | ⭐⭐⭐⭐ |
| **智谱 AI** | GLM-4 | 有 | ⭐⭐⭐⭐⭐ |
| **Moonshot** | Kimi | 有 | ⭐⭐⭐⭐ |
| **DeepSeek** | DeepSeek-V2 | 有 | ⭐⭐⭐⭐⭐ |
| **OpenAI** | GPT-4 | $5 试用 | ⭐⭐⭐ |
| **Anthropic** | Claude | 有 | ⭐⭐⭐ |

**我推荐：** 智谱 AI 或 DeepSeek，中文支持好，免费额度够用。

### 4.2 获取 API Key

**以智谱 AI 为例：**

1. 访问：https://open.bigmodel.cn/
2. 注册账号（手机号即可）
3. 实名认证（必须，很快）
4. 进入控制台：https://open.bigmodel.cn/console/apikey
5. 创建 API Key
6. 复制保存（只显示一次）

**免费额度：** 新用户送价值 ¥25 的 token，够用很久。

### 4.3 配置到 OpenClaw

运行配置命令：

```bash
openclaw configure --section model
```

按提示输入：
- 模型提供商（如 `zhipu`）
- API Key（刚才复制的）
- 模型名称（如 `glm-4`）

**或者手动编辑配置文件：**

位置：`~/.openclaw/config.json`

```json
{
  "model": {
    "provider": "zhipu",
    "apiKey": "你的 API Key",
    "modelName": "glm-4"
  }
}
```

### 4.4 测试模型

```bash
openclaw chat "你好，测试一下"
```

如果有回复，说明配置成功！

## 十二、遇到的问题及解决

### 12.1 Browser Relay 连接失败

**现象：**
```
Error: Chrome extension relay is running, but no tab is connected.
```

**原因：**
- Gateway 没启动
- Chrome 扩展没连接

**解决：**
```bash
openclaw gateway start
```

然后在 Chrome 中打开页面，点击扩展图标。

### 12.2 clawhub 速率限制

**现象：**
```
Rate limit exceeded
```

**原因：**
ClawHub 有请求频率限制。

**解决：**
- 等 5-10 分钟再试
- 或手动下载技能包

### 12.3 Tavily API Key 配置

**现象：**
```
Missing TAVILY_API_KEY
```

**解决：**
```bash
$env:TAVILY_API_KEY="你的 key"
```

或运行：
```bash
openclaw configure --section tavily
```

### 12.4 模型 API 额度用完

**现象：**
```
API quota exceeded
```

**解决：**
- 方案 A：等新周期（每月 1 号重置）
- 方案 B：充值（一般 ¥100 够用很久）
- 方案 C：换其他提供商（智谱、DeepSeek 等）

## 十三、AI 变现思路

搭建小墨的过程中，我发现了几个变现方向：

### 13.1 AI 助手 SaaS

**思路：** 把小墨封装成产品，收费订阅。

**定价：**
- 基础版：¥9.9/月（基础功能）
- 专业版：¥29.9/月（全部功能）
- 企业版：¥99/月（多用户、定制）

**技术栈：**
- 前端：Next.js + Vercel
- 后端：OpenClaw + 数据库
- 支付：Stripe/支付宝/微信

**获客：**
- CSDN、知乎写教程引流
- 社交媒体运营
- SEO 优化

### 13.2 垂直领域助手

**思路：** 专注某个领域，做深做精。

**方向举例：**

| 领域 | 功能 | 目标用户 |
|------|------|----------|
| **开发者助手** | 代码审查、Bug 修复、部署 | 程序员 |
| **学生助手** | 作业辅导、考试复习、留学申请 | 学生 |
| **投资人助手** | 股票分析、研报整理、新闻推送 | 投资者 |
| **电商助手** | 选品分析、竞品监控、客服 | 电商从业者 |
| **自媒体助手** | 选题策划、内容生成、数据分析 | 自媒体人 |

**优势：**
- 竞争小
- 用户付费意愿强
- 容易建立壁垒

### 13.3 教程/培训

**思路：** 教别人搭建 AI 助手。

**形式：**
- 文字教程（CSDN、知乎、公众号）
- 视频课程（B 站、YouTube）
- 直播培训（腾讯会议）
- 1 对 1 咨询

**定价：**
- 免费教程（引流）
- 付费课程：¥99-999
- 企业内训：¥5000-50000

**我的案例：**
就是你现在看的这篇文章！发布到 CSDN 后可以：
- 引流到私域
- 接广告
- 卖课程
- 接咨询

### 13.4 技能开发

**思路：** 开发 Skills 放到 ClawHub 售卖。

**技能举例：**
- 特定 API 集成（如Notion、飞书）
- 垂直领域工具（如股票分析、法律检索）
- 自动化工具（如定时任务、工作流）

**定价：**
- 免费技能（引流）
- 付费技能：¥9.9-99.9

**分成：**
ClawHub 有分成机制，开发者可以赚钱。

### 13.5 自媒体内容

**思路：** 分享 AI 使用技巧，引流变现。

**平台：**
- CSDN（技术文章）
- 知乎（问答 + 专栏）
- 公众号（深度文章）
- B 站（视频教程）
- 小红书（使用技巧）

**变现方式：**
- 平台分成（流量主）
- 广告收入
- 付费社群
- 知识星球
- 带货（AI 工具、课程）

## 十四、总结

### 14.1 核心要点

搭建个人 AI 助手的门槛比想象中低很多：

- **技术门槛：** 会基本的命令行操作即可
- **时间成本：** 1 天可以上线基础版
- **资金成本：** 几乎为零（用免费额度）

### 14.2 关键步骤

1. 安装 OpenClaw
2. 配置 AI 模型
3. 定制人设
4. 安装 Skills
5. 配置自动任务
6. 添加实用功能
7. 绑定聊天渠道

### 14.3 最重要的建议

**现在就开始动手！**

不要等"准备好了"再开始，因为：
- 你永远准备不好
- 边做边学最快
- 完成比完美重要

### 14.4 小墨的成长

小墨不是一天建成的，我会持续优化：

- 添加更多实用功能
- 优化人设和对话体验
- 探索变现模式
- 分享经验和教训
